# 🌐 Check-It
> `CHECK-IT` is a link aggregator site, similar to Reddit or Hacker News


![Screenshot](./app.gif)

## Getting Started 🚀

* ``` git clone https://github.com/ragmha/check-it.git```

* ``` rvm install ruby-2.2.2 ```

* ``` sudo gem install rails ```

* ``` rails s  ```
